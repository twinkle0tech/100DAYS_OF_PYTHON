##  What This Code Does

- Moves the robot forward until it finds a wall  
- Turns left to start solving the maze  
- Uses the **Right-Hand Rule Algorithm**:
- If the right side is clear ‚Üí turn right and move  
- Else if front is clear ‚Üí move forward  
- Else ‚Üí turn left
