# Silent Auction | Python Project

This is a simple Python project that simulates a silent auction system.
Users can enter their name and bid amount, and the program will finally display the highest bidder.

---

##  Features

- Accepts multiple bidders

- Stores all bids in a dictionary

- Finds the highest bid

- Displays the winner at the end

---

##  Requirements

- Python 3.x

- art module (used for the logo)

- Install the module using:

` pip install art `

---

##  How to Run

- Save the file as:

  ` main.py `


- Run the program:

  ` python main.py `

---

## Program Flow

- Program shows a logo

- Asks user for:

   -> name
   -> bid amount

- Asks if there are more bidders

- When bidding ends, it prints the winner
