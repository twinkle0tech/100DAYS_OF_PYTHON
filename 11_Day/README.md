# Blackjack Game (Python)

This is a simple **Blackjack card game** built using Python.
The player plays against the computer.

---

##  Features

- Random card dealing
- Automatic score calculation
- Blackjack detection
- Computer plays automatically
- Simple terminal based game

---

##  Files used

- `main.py` → main game file  
- `art.py` → contains the game logo

---

##  Rules

- If your score is **21 with two cards**, it is a Blackjack.
- If your score goes above **21**, you lose.
- The computer will draw cards until its score is at least 17.

- Python
- random module
